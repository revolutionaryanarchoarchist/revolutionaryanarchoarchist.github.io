1. Setting Up the Test

This is a one‑sample t‑test problem. We want to check whether the true mean weight of whitefish could be μ₀ = 3 kg, based on the sample of 8 fish:

Data: 3, 2, 4, 1, 9, 3, 1, 2

Hypotheses:

  • H₀: μ = 3
  • H₁: μ ≠ 3

Step 1 — Sample Mean

xˉ=83+2+4+1+9+3+1+2​=825​=3.125

Step 2 — Sample Variance

t=s/n​xˉ−μ0​​=2.588/8​3.125−3​=0.9150.125​≈0.137

Step 4 — Compare with Critical Value

Degrees of freedom: df = n − 1 = 7

We fail to reject H₀. The data give no evidence against μ = 3; the discrepancy between the sample mean (3.125) and 3 is well within what's expected from random sampling variation. So it is reasonable to assume the population mean weight of the whitefish equals 3 kg.

p=31​,q=1−p=32​

If a student guesses randomly on all 20 questions, there is only about a 1.76% chance of getting 2 or fewer questions correct — meaning random guessing usually leads to more than 2 correct answers (the expected number is np=20/3≈6.67). 3. Setting Up the Test

This is a Chi-square test of independence, used to test whether two categorical variables (leadership style and performance level) are associated.

Hypotheses:

  • H₀: Leadership style and performance are independent (no association)
  • H₁: Leadership style and performance are associated
